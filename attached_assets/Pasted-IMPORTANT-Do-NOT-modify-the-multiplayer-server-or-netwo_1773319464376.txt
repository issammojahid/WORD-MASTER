1) COIN ENTRY MATCH SYSTEM

Online matches should no longer be free.

Players must select a coin entry before starting a match.

Example entry options:

- 50 coins → winner receives 100 coins
- 100 coins → winner receives 200 coins
- 500 coins → winner receives 1000 coins
- 1000 coins → winner receives 2500 coins

Flow:

1. Player selects entry amount
2. Matchmaking finds another player with the same entry
3. Both players pay the entry coins
4. Winner receives the reward

Coins must be deducted only after a match is successfully created.
